## Instalación

Para implementar este proyecto, ejecute

```bash
  git clone https://github.com/cybertcode/FRONTEND-BLOG-TAILWINDCSS.git
```
Entrar en el directorio y ejecutar
```bash
  npm install && npm run dev
```
